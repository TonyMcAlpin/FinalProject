This converter can be used for many different conversions that you may run into while following recipes when cooking and baking. Almost every recipe out there will provide you with one unit for each ingredient. It is very common to not have a measuring device on hand that has that exact unit. Using this site you will easily be able to convert the given unit to a unit that you have available to use.


Example: You are cooking steak and you all following a recipe for a marinade. The recipe calls for 1.5 pints of water, but you need this measurement in cups. This can easily be done right here. Start by entering 1.5 into the first text box below, and then select pints in the box labeled “Select Your Starting Unit”. Then select cups in the last box labeled “Select Your Starting Unit”. Upon selecting cups, the conversion will automatically happen, and the number 3.0 will appear in the third box.
